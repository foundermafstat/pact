# Pact Threat Model

| Threat | Mitigation | Test or script proving mitigation | Known limitation |
| --- | --- | --- | --- |
| Replayed milestone proof | Milestone nullifier is stored and repeat submit fails after release. | `pnpm demo:attacks` checks replay rejection; `cargo test -p milestone-escrow` covers reused nullifier and same tranche twice. | API mock submit currently returns `tranche_release_failed` after release rather than an on-chain nullifier code. |
| Revoked credential reused | Revoked credentials are rejected by proof generation and root rotation omits revoked credentials. | `pnpm --filter @pact/api test` covers `credential_not_active`; issuer root tests cover revoked credential omitted from new root. | Production revocation depends on timely root publish/indexing. |
| Cross-program replay | Milestone proof public inputs must match target program and milestone. | `pnpm demo:attacks` checks `milestone_public_inputs_mismatch`; milestone circuit tests bind program/milestone into nullifier. | Real Poseidon binding is represented by deterministic placeholder arithmetic until hardening. |
| Wrong recipient payout | API and contract compare proof recipient against tranche release wallet. | `pnpm demo:attacks`; `cargo test -p milestone-escrow` wrong recipient test. | UI attack panel is mocked for demonstration. |
| Wrong tranche amount | Contract and circuit bind tranche amount. | `cargo test -p milestone-escrow`; `pnpm zk:test:milestone` wrong amount fixture. | API submit validates amount through stored proof public inputs. |
| Inactive or revoked root | Root status must be active before proof input/proof acceptance. | `pnpm demo:attacks` inactive root case; contract inactive root tests. | In-memory services model root state for MVP; production requires DB-backed root registry sync. |
| Hidden KPI disclosure | Public audit projection strips private metrics and frontend privacy guards deny private fields. | `pnpm --filter @pact/web test`; `pnpm --filter @pact/api test` public audit tests. | Attestor console necessarily accepts raw KPI input from the attestor role. |
| Credential secret disclosure | API client and UI do not render credential secrets in public routes. | `pnpm --filter @pact/web test`; `pnpm --filter @pact/api test` issuer proof tests. | Mock issuer returns private package to issuer-only flow in MVP. |
| Invalid eligibility policy | Eligibility circuit enforces sanctions, expiry, accredited or non-US, policy binding, and nullifier context. | `pnpm zk:test:eligibility`. | Placeholder Merkle/Poseidon constraints are not production cryptography. |
| Event/audit duplication | Indexer cursor store resumes from last processed ledger and sink deduplicates mapped events by tx/event. | `pnpm --filter @pact/indexer test`. | RPC polling implementation is skeleton; real Stellar RPC pagination is pending deployment hardening. |
